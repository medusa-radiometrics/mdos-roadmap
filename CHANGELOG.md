# Changelog

All notable changes to mDOS are tracked in this file, per version.

## [v4.0 - build 7] - 2026-07-23

### Fixed
- Corrected default device configuration.
- Improved handling of invalid MCF files to prevent loading errors.

## [v4.0 - build 6] - 2026-07-20

### Added
- Added support for the Medusa Lidar sensor.

## [v4.0 - build 5] - 2026-07-14

### Added
- First public release of mDOS V4.0 See the [mDOS V4.0 Manual](https://links.medusa-radiometrics.com/mdos-manual) for full documentation.
